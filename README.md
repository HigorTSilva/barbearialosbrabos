# ✂️ Barber — Landing Page para Barbearia

> Landing page institucional e funcional para barbearia. Apresenta os serviços com preços, localização via Google Maps, redes sociais e botão flutuante de contato direto pelo WhatsApp.

## ✨ Funcionalidades

- 🎯 **Hero section** com slogan, horário de funcionamento e CTA "Agendar Horário"
- 💈 **Seção de Serviços** — cards com foto, nome e preço de cada serviço
- 📖 **Seção Sobre** — galeria de fotos e descrição da barbearia
- 📍 **Mapa interativo** com localização via Google Maps embarcado
- 💬 **Botão flutuante do WhatsApp** fixo na tela ao descer um pouco o scroll — redirecionamento direto para conversa
- 📱 Links para **Instagram, Facebook e TikTok** no header e footer
- 📱 Layout **responsivo** para mobile e desktop

---

## 📋 Seções da Página

| Seção | Descrição |
|---|---|
| **Header / Hero** | Logo, redes sociais, slogan, horário e botão de agendamento |
| **Sobre** | Galeria de fotos em collage e texto institucional |
| **Serviços** | Cards com foto, nome e preço (Corte Normal, Barba Completa, Barba e Cabelo) |
| **Venha Conhecer** | Google Maps embarcado com o endereço da unidade |
| **Footer** | Ícones de redes sociais e logo |
| **WhatsApp (fixo)** | Botão flutuante de contato rápido |

---

## 🛠️ Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — layout responsivo, collage de imagens e animações
- **JavaScript (Vanilla)** — comportamentos de UI e scroll suave
- **Google Maps Embed API** — mapa interativo sem necessidade de chave de API
- **WhatsApp API (`wa.me`)** — redirecionamento direto para conversa no WhatsApp
