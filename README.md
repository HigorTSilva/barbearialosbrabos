## 📌 Sobre o Projeto

Esta é uma **landing page estática e institucional**. Seu objetivo é apresentar a marca Churras Fast, destacar as unidades disponíveis e direcionar o visitante para as próximas etapas (cardápio e delivery), que podem ser implementadas futuramente.

> ⚠️ **Os botões "Pedir pelo Delivery" e "Saiba mais sobre o Rodízio" ainda não possuem funcionalidade implementada.** Estão presentes apenas como elementos visuais de chamada para ação (CTA).

---

## ✨ O que a página apresenta

- 🥩 **Hero section** com foto de destaque, slogan e CTAs principais
- 🏪 **Seção de unidades** — endereços e informações de cada restaurante
- 📋 **Chamada para o cardápio** (sem funcionalidade no momento)
- 🛵 **Chamada para delivery** (sem funcionalidade no momento)
- 📱 Layout responsivo para mobile e desktop

---

## 🗺️ Roadmap

Funcionalidades planejadas para versões futuras:

- [ ] Integração com cardápio digital
- [ ] Botão de delivery funcional (redirecionamento para WhatsApp ou plataforma de pedidos)
- [ ] Seção "Saiba mais sobre o Rodízio" com detalhes e preços
- [ ] Formulário de reserva de mesa
- [ ] Galeria de fotos do restaurante
- [ ] Integração com Google Maps para localização das unidades
- [ ] SEO otimizado (meta tags, Open Graph)
